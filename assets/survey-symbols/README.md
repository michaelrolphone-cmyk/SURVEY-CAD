# Survey Symbol Library
Standardized SVG symbols for common survey point-file codes and map annotations.

## Usage
- View all files in `assets/survey-symbols/`.
- Use `assets/survey-symbols/index.json` as a machine-readable manifest.
- Each symbol uses a `0 0 100 100` viewBox and monochrome strokes so they can be recolored by CSS/SVG processing.
- Text-based symbols use short utility codes (e.g., `WM`, `GM`, `SMH`, `BM`).
